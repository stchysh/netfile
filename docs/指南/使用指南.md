# NetFile 使用指南

## 快速开始

### 1. 启动服务

```bash
# 使用默认配置启动
netfile start

# 指定实例名称
netfile start --name MyServer

# 使用自定义配置文件
netfile start --config /path/to/config.toml
```

### 2. 查看在线设备

```bash
# 列出所有在线设备
netfile devices list

# 查看特定设备信息
netfile devices info <instance_id>

# JSON 格式输出
netfile devices list -o json
```

### 3. 发送文件

```bash
# 发送单个文件
netfile send 192.168.1.100:37050 /path/to/file.txt

# 发送文件夹（递归）
netfile send 192.168.1.100:37050 /path/to/folder --recursive

# 启用压缩
netfile send 192.168.1.100:37050 large_file.zip --compress
```

### 4. 查看传输进度

```bash
# 列出所有传输任务
netfile transfers list

# 查看特定传输详情
netfile transfers info <transfer_id>
```

## TUI 终端界面

### 启动 TUI

```bash
# 使用默认配置
netfile-tui

# 指定实例名称
netfile-tui --name Server1
```

### TUI 操作

- **Tab** / **Shift+Tab**: 切换标签页
- **1**: 切换到设备列表
- **2**: 切换到传输队列
- **3**: 切换到设置
- **q**: 退出

### TUI 标签页说明

#### Devices（设备列表）
显示局域网内所有在线设备，包括：
- 设备名称
- 实例名称
- IP 地址和端口
- 自动刷新（1秒间隔）

#### Transfers（传输队列）
显示当前传输任务，包括：
- 文件名和进度百分比
- 进度条
- 传输速度（MB/s）
- 自动刷新（500ms间隔）

#### Settings（设置）
显示当前配置信息：
- 实例信息
- 网络配置
- 传输配置
- 安全配置

## 配置管理

### 查看配置

```bash
# 显示当前配置
netfile config show

# JSON 格式
netfile config show -o json
```

### 修改配置

```bash
# 设置实例名称
netfile config set instance.instance_name "My Server"

# 启用压缩
netfile config set transfer.enable_compression true

# 设置最大并发数
netfile config set transfer.max_concurrent 5

# 启用 TLS
netfile config set security.enable_tls true
```

### 重置配置

```bash
# 重置为默认配置
netfile config reset
```

## 安全功能

### 设备授权

```bash
# 列出授权设备
netfile auth list

# 授权设备
netfile auth allow <device_id>

# 取消授权
netfile auth deny <device_id>
```

### 密码保护

```bash
# 设置密码
netfile auth set-password

# 发送文件时提供密码
netfile send 192.168.1.100:37050 file.txt --password "your_password"
```

### TLS 加密

```bash
# 启用 TLS
netfile config set security.enable_tls true

# 重启服务使配置生效
netfile restart
```

## 高级功能

### 断点续传

传输中断后，重新发送相同文件会自动从断点继续：

```bash
# 第一次传输（中断）
netfile send 192.168.1.100:37050 large_file.zip

# 重新传输（自动续传）
netfile send 192.168.1.100:37050 large_file.zip
```

### 压缩传输

对于可压缩文件，启用压缩可以提升传输速度：

```bash
# 启用压缩
netfile send 192.168.1.100:37050 logs.txt --compress

# 或在配置中全局启用
netfile config set transfer.enable_compression true
```

### NAT 穿透

NetFile 支持通过 STUN 和 UDP 打洞实现 NAT 穿透：

```bash
# 启用 NAT 穿透
netfile config set network.enable_nat_traversal true

# 设置 STUN 服务器
netfile config set network.stun_servers "stun.l.google.com:19302"
```

## 使用场景

### 场景 1: 局域网文件共享

```bash
# 设备 A: 启动服务
netfile start --name DeviceA

# 设备 B: 启动服务并查看设备
netfile start --name DeviceB
netfile devices list

# 设备 B: 发送文件到设备 A
netfile send 192.168.1.100:37050 document.pdf
```

### 场景 2: 大文件传输

```bash
# 启用压缩和多线程
netfile config set transfer.enable_compression true
netfile config set transfer.max_concurrent 5

# 发送大文件
netfile send 192.168.1.100:37050 movie.mkv
```

### 场景 3: 文件夹同步

```bash
# 递归发送整个文件夹
netfile send 192.168.1.100:37050 /path/to/project --recursive

# 接收端会保留完整的目录结构
```

### 场景 4: 监控传输状态

```bash
# 终端 1: 启动 TUI 监控
netfile-tui --name Monitor

# 终端 2: 执行文件传输
netfile send 192.168.1.100:37050 large_file.zip

# 在 TUI 的 Transfers 标签页实时查看进度
```

## 常见问题

### Q: 如何找到接收端的 IP 和端口？

**A**: 使用 `netfile devices list` 命令查看所有在线设备的地址。

### Q: 传输速度慢怎么办？

**A**:
1. 启用压缩：`--compress` 参数
2. 增加并发数：`netfile config set transfer.max_concurrent 10`
3. 增加块大小：`netfile config set transfer.chunk_size 2097152`

### Q: 如何确保传输安全？

**A**:
1. 启用 TLS 加密：`netfile config set security.enable_tls true`
2. 设置密码保护：`netfile auth set-password`
3. 启用设备授权：`netfile config set security.require_auth true`

### Q: 传输中断后如何恢复？

**A**: 重新执行相同的发送命令，NetFile 会自动检测并从断点继续传输。

### Q: 如何在不同网络间传输文件？

**A**:
1. 启用 NAT 穿透：`netfile config set network.enable_nat_traversal true`
2. 或使用端口转发/VPN 等方式建立连接

### Q: 接收的文件保存在哪里？

**A**:
- Windows: `C:\Users\<用户名>\Downloads\NetFile\`
- Linux/macOS: `~/Downloads/NetFile/`
- 可通过配置文件修改下载目录

详细命令说明请使用 `netfile <command> --help` 查看。

## 配置文件说明

配置文件位于 `~/.netfile/config.toml`：

```toml
[instance]
instance_id = "uuid"           # 实例唯一标识
instance_name = "My Server"    # 实例名称
device_name = "PC-NAME"        # 设备名称

[network]
discovery_port = 37020         # 发现端口
transfer_port = 37050          # 传输端口
broadcast_interval = 5         # 广播间隔（秒）

[transfer]
chunk_size = 1048576          # 块大小（字节）
max_concurrent = 3            # 最大并发数
enable_compression = false    # 启用压缩

[security]
require_auth = true           # 需要授权
password = ""                 # 密码（SHA256哈希）
allowed_devices = []          # 授权设备列表
enable_tls = false           # 启用 TLS
```

## 性能调优

### 提升传输速度

```bash
# 增加块大小（适合高速网络）
netfile config set transfer.chunk_size 4194304

# 增加并发数
netfile config set transfer.max_concurrent 10

# 启用压缩（适合文本文件）
netfile config set transfer.enable_compression true
```

### 降低内存占用

```bash
# 减少块大小
netfile config set transfer.chunk_size 524288

# 减少并发数
netfile config set transfer.max_concurrent 1
```

### 降低网络占用

```bash
# 增加广播间隔
netfile config set network.broadcast_interval 10

# 禁用压缩（减少 CPU 占用）
netfile config set transfer.enable_compression false
```
